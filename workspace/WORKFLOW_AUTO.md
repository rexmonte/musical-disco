## WORKFLOW_AUTO.md

### System Workflow Automation
- **Auto-Triggered Tasks:**
  - Memory compaction audits
  - Model performance monitoring
  - Skill integration checks
- **Default Protocols:**
  - Read required startup files after context reset
  - Append memory entries without overwriting
  - Validate skill configurations
- **Error Handling:**
  - Retry failed audits with delay
  - Log compaction states
  - Notify on configuration conflicts
